### /read_url

**Purpose:** To fetch and read the content of a public URL.

**Usage:** `/read_url "https://docs.anthropic.com/en/docs/build-with-claude/pdf-support"`

**Description:**
This command is used for research and information gathering. The Planner Agent can use it to consult external documentation, API references, or articles on best practices to ensure that the plan is based on the most current and effective methods.
