### /ask

**Purpose:** To ask clarifying questions to the user or other agents to ensure a clear understanding of a task or problem.

**Usage:** `/ask "[Question to be clarified]"`

**Description:**
This command is used to gather necessary information before starting the planning process. The Planner Agent will use this to resolve ambiguities, fill in missing details, or confirm assumptions. The planning process will be paused until a satisfactory answer is provided.
