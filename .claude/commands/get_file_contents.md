### /get_file_contents

**Purpose:** To retrieve the content of a specific file for detailed analysis.

**Usage:** `/get_file_contents "[Path to the file, e.g., 'src/models/user.js']"`

**Description:**
This command is a fundamental tool for the Planner Agent. It allows the agent to read and understand the actual code or configuration of a file, which is crucial for accurate planning, dependency mapping, and risk assessment. The output is the raw content of the specified file.
