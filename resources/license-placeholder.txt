SMC License Placeholder
========================

This file indicates where license files should be placed.

Production Deployment Steps:
1. Generate license file using: smc-license generate [options]
2. Copy license file to this resources/ directory  
3. Replace this placeholder file
4. Test license activation with SMC application

License File Requirements:
- File name: license.lic OR license.json (exact case)
- Priority: license.json takes precedence over license.lic if both exist
- Format: AES-256-CBC encrypted JSON (for .lic) or plain JSON (for .json)
- Contains: organization, customer, MAC address, expiry date
- Generated by: SMC License CLI tool
