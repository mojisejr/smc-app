### Context Optimization Scripts
```bash
# validate-context.sh
#!/bin/bash
ROUND=$1
SUPPLEMENT=$(grep "Round${ROUND}:" .context-map | cut -d: -f2)
TOKENS=$(wc -w CLAUDE.md ${SUPPLEMENT} | tail -1 | cut -d' ' -f1)

if [ $TOKENS -gt 9000 ]; then
  echo "⚠️  Context too large: ${TOKENS} tokens"
  echo "💡 Optimize ${SUPPLEMENT} before proceeding"
  exit 1
else
  echo "✅ Context size optimal: ${TOKENS} tokens"
fi
```
